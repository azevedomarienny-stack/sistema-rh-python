# Sistema de Gerenciamento de Funcionários em Python

Projeto desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

## Sobre o projeto

Este sistema simula um pequeno gerenciamento de funcionários, permitindo cadastrar informações profissionais e calcular salários de acordo com as horas trabalhadas.

O objetivo principal do projeto foi praticar conceitos fundamentais de Python e melhorar minha lógica de programação através de uma aplicação mais próxima de um sistema real.

---

## Funcionalidades

* Cadastro de funcionários
* Geração automática de matrícula
* Cadastro de:

  * Nome
  * Email
  * Celular
  * CPF
  * Cargo
  * Departamento
  * Data de contratação
* Registro de horas trabalhadas
* Cadastro de salário por hora
* Cálculo salarial
* Busca de funcionários por nome
* Busca de funcionários por matrícula
* Menu interativo no terminal
